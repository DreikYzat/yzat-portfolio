Projeto em Destaque

📦 YZAT Almoxarifado

Sistema completo de gerenciamento de almoxarifado e controle de estoque, desenvolvido com aplicação Web, API REST, banco de dados PostgreSQL online e versão Android APK.

O sistema possui login com autenticação, controle de usuários por cargo, cadastro de produtos, entrada e saída de estoque, histórico de movimentações, dashboard com gráficos, pesquisa em tempo real e exportação de relatórios em PDF.
